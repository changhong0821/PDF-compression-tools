# PDF-compression-tools
這是一款基於 Python 開發的高效能 PDF 處理工具，專為處理 超大型文件 (如 600MB+) 設計。具備智能採樣預判功能，能精準將 PDF 壓縮至目標容量。

✨ 核心特色
⚡ 多核心加速：利用 Multiprocessing 技術，處理速度提升 3-8 倍。
🃏 洗牌式採樣：自動抽取文件 20% 頁面分析，精準鎖定最佳壓縮參數。
🛡️ 容量硬性限制：輸入 200MB 就絕不會產出 201MB，具備二次重壓保護。
📸 全格式支援：支援 PDF 壓縮、JPG/PNG 合併，甚至是 iPhone 的 HEIC 格式。
🛠️ 全能工具：整合壓縮、合併、拆分、加密、浮水印、頁面刪除等 8 大功能。

功能選單預覽：
壓縮 PDF：輸入目標 MB 數，程式自動計算最佳畫質。
圖片轉 PDF：支援批量資料夾匯入（含 HEIC）。
合併/拆分：快速整合或拆解頁面。
安全加密：設定 AES-256 開啟密碼。
小撇步：在 Windows 上使用時，可以直接將檔案或資料夾「拖入」視窗來輸入路徑。

⚠️ 注意事項
本工具採用有損壓縮技術，壓縮後的文字將會轉為圖片型態（無法選取文字）。
處理超大型檔案時，請確保電腦有足夠的 RAM 空間。
